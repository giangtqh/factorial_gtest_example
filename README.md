# factorial_gtest_example

## Install googletest library on you linux host
```
$ git clone https://github.com/google/googletest.git
$ cd googletest
$ git checkout tags/v1.10.x -b v1.10.x
$ cd ..
$ mkdir buildgtest
$ cd buildgtest
$ cmake ../googletest
$ make
$ sudo make install
```
